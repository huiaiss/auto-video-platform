三种UI架构方案对比

目标：简单上手 + 批量出片 + 每步可预览。看哪个架构最对胃口？

A

流水线向导 — Pipeline Wizard

像"做菜食谱"一样，一步一步走：上传图 → 确认脚本 → 试听配音 → 预览画面 → 一键渲染。最有安全感，不会出错。

适合：第一次用、不确定参数、想每步确认

B

仪表盘 + 快速生成 — Dashboard

左边配置栏、右边大"生成"按钮 + 结果展示区。可以一次填多个选题排队批量出片。效率最高。

适合：熟悉流程、批量操作、追求效率

C

混合标签页 — Hybrid Tabs（推荐）

顶部四个标签：快速生成（向导模式）| 批量队列 | 历史记录 | 设置。一个新用户进来默认走向导，老手切到批量模式。参考了MoneyPrinterTurbo（5.5万⭐）的成熟架构。

适合：兼顾新手和老手、未来迭代空间大
